Note
Define LOGIC_OUT_IT shared symbol at project level to use GPIO_out from timer interrupts (for more timing precision if required)
When using GPIO_out from interrupts, shared LOGIC_OUT_IT_PER period is defined by default with a period of 1000Âµs (can be customly defined)
Warning
